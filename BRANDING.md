# GhostPilot AI — Product One-Liner

An AI browser co-pilot that watches websites, understands your intent, suggests the safest next action, and turns repeated browser work into reusable workflows.

## Core Screens

- Landing page: product story + futuristic browser hero
- Dashboard: AI task input + workflow cards
- Browser activity panel: simulated live page reading
- AI suggestion panel: real-time typing effect
- Automation history: completed action log
